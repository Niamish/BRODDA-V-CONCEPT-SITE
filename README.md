# BRODHA V — Dark Grunge Artist Website

A bold, cinematic, dark-grunge artist website concept for **Brodha V**, built around the energy of Indian hip-hop, mythology, street culture, and underground performance visuals.

The site uses glitch animations, noise overlays, scanlines, red-gold accents, bold typography, animated music cards, video highlights, tour listings, and responsive navigation to create a raw concert-poster style digital experience.

## Features

- Full-screen animated loading screen
- Dark grunge visual identity
- Glitch text animation
- Noise texture and scanline overlay
- Responsive navigation menu
- Hero section with bold Brodha V typography
- About section with artist story and stats
- Music discography grid
- Video showcase layout
- Tour dates section
- Social media footer
- Cursor trailer effect
- Smooth scrolling
- Scroll-based hero parallax
- Intersection animations for cards and tour items

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts

## Sections

### Hero

A dramatic full-screen intro featuring **BRODHA V**, glitch typography, red accent visuals, and the tagline:

**Mythology × Hip-Hop × Street**

### About

Introduces Brodha V as an Indian rapper and music producer from Bangalore, highlighting his fusion of mythology, multilingual lyricism, and modern hip-hop culture.

### Music

A discography-style grid featuring tracks such as:

- Aathma Raama
- Aigiri Nandini
- Shiva Tandava Stotram
- Vainko
- Indian Flava
- After Party

### Videos

A featured video section with a main video panel and supporting side thumbnails.

### Tour

Upcoming show listings with dates, venues, cities, and ticket buttons.

### Contact

Social links and footer branding for artist connection and updates.

## How to Run

Open the HTML file directly in your browser:

```bash
brodha-v-dark-grunge.html
```

No build tools, npm packages, or server setup are required.

## Customization

You can easily change:

* Artist name
* Track list
* Tour dates
* Social links
* Color palette
* Hero tagline
* About text
* Video thumbnails
* Footer credits

Most of the styling can be edited through the CSS variables inside the `:root` selector.

## Responsive Design

The website adapts across desktop, tablet, and mobile devices. On smaller screens, the navigation becomes a fullscreen mobile menu.
